      JSON 데이터로 관리되며, 레이아웃 개편 시에도 콘텐츠 구조를 재사용할 수 있습니다.

      확장 가능한 운영 구조

      • 현재: 정적 페이지 + JSON 콘텐츠 + GitHub Pages 자동 배포
      • 다음: 무료 Supabase를 붙여 독자 피드백/좋아요 저장
      • 그다음: Cloudflare Workers로 간단 API/자동화 엔드포인트 연결
      • 수익화: 작품별 후원 링크, 구독 전환 랜딩, 광고 슬롯 추가 가능
